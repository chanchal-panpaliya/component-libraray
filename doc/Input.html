HTML Forms are used to send data across the web and The input element is used to get user information .
The <input > HTML element is used to create interactive controls for web-based forms in order to accept data from the user. The <input > tag specifies an input field where the user can enter data. The input element is used to get user information. An input element can be of type text, password, checkbox, radio button, submit button and more

Input type "text" is the common input element for name, surname, country, numbers, and symbols in a single line. The default input type is always text. Maxlength
maxlength attribute is used to restrict no of characters in a text field. maxlength value is a number. Maxlength attribute is useful for form validations.
< input type="text" id="fname" maxlength="10" >
Size
Size attribute is used to set the size of input or textarea. The default size is 20. To increase or decrease input size, change value of size
< input type="text" size="3" >
