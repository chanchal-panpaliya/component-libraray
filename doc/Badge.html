Easlily create nicely looking badges to label and highlight your content.
Badges is used to add additional information to the content. For example, some websites associate some number of notifications to the link. The notification number is seen when logged in to a particular website which tells the numbers of news or notifications to see by clicking it. Badges can be used inside buttons, tables, paragraphs, lists, etc. We can even color them or change their size.
Note : Badge font-size % is based on the parent font-size class. Here we are using 75% as a default font-size of the parent class.
To create a badge, use the .badge class and also add the badge colour and font-size. Badge colour and Font-Size.
