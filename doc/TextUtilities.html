The text utility contains helper classes for the CSS text align property and text utilities to control alignment, wrapping, weight, and more.
All text elements are written inside body. < p > is the common tag use to write paragraphs. There are six headings in html, <h1 > , <h2 > , <h3 > , <h4 > , <h5 > and <h6 > . By default the plain text is black in color and with transparent background. To change color and backgrounds, we need to use color table .
