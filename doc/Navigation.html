A Navigation menu is an organized list of links to other web pages.
A set of buttons or images in a row or column that serves as a control point to link the user to sections on a website. The navigation bar may also be a single graphic image with multiple selections Navigation menus appear most commonly in page headers or sidebars across a website, allowing visitors to quickly access the most useful pages. A navigation menu is an organized list of links to other web pages, usually internal pages.
The horizontal navigation bar is the most common type of navigation menu. It lists the major pages side-by-side and is placed in the website header. For the Horizontal Navigation Bar, add .horizontal-nav-header classname to the <Header > tag and for the <ul> tag, add horizontal-nav-list class name.
