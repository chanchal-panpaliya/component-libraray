The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.
Every page layout should fit into and utilize the common grid strategy. The grid consists of several key parts that include the Container, Container Margins, Columns, and Gutters. All columns reside within the Container, which is located at the root level of a page. Depending on the breakpoint, the container will have Container Margins of various sizes. The container also has a maximum width property, beyond which only the container margins grow in size.
This component defines a set of standards and guidelines for laying out pages. A page may have a visual layout that divides content into sections or "regions" (i.e., a header, footer, navigation, etc.) HTML5 has created tags for these sections of a page. Add <display: grid> for to display items are placed in rows by default and span the full width of the grid container. Explicitly set a grid by creating columns and rows with the grid-template-columns and grid-template-rows properties.
