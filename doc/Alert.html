Defines styles for success, warning and error messages.

An alert displays a short , important message in a way that attracts the user's attention without interrupting the user's task. Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages. Alerts are available for any length of text , as well as an optional dismiss button. Alerts are available in 5 types- success alert, error alert, warning alert, primary alert, secondary alert.
