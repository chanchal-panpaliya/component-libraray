Card are used to show user related data collectively, like product details.
Cards are used to show collective information like product price, preview, and other details. A card is a content container and includes options for headers Icon and Image, a wide variety of content, icon background colors, and powerful shapes. Use .card-body as a parent class name. In the .card-body class, apply .flex-col class to wrap the .card-heading class, sample heading text, and card description.
For Icon card Use .card-body as a parent class name.Wrap the .card-heading class, sample heading text, and card description using the .flex-col class. The e-commerce product card with a badge is used to show the product card with the discount text. For this card, we're using a different shape of badge. Use .Card-with-badges-body as a parent class along with sub class .card-header for header . Inside the header use .card-image-header for the image. also add diffrent shape of badge class inside -- .shape-product-today , .shape-product-today-text , .top-right-vertical-shape , .top-right-vertical-shape-text , .top-right-horizontal-shape , .top-right-horizontal-shape-text , .discount-shape , .discount-shape-text , .top-left-horizontal-shape , .top-left-vertical-shape . To add these shapes, add your.card-header class position as a relative.

Text only cards are simple cards with only text as content and a fixed width. By default, a card's width has no fixed width, so it'll naturally fill the full width of its parent element. Text-only cards are similar to basic cards. Here we are using the same class name as the basic card. use .card-body for a based class and .card-header as a sub class.
