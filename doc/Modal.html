A modal is a dialog box/popup box that is displayed on top of the current page
Modals display content that temporarily blocks interactions with the main view of the UI. Use modals sparingly to avoid unnecessarily interrupting the users workflow. Modal is use for - Confirming or making decisions , Providing additional information , Completing important tasks in a workflow, Asking for feedback; e.g., "How would you rate this tutorial?"
If a modal doesn't require users to take an action and the content isn't critical, then you don't need to use buttons. Users can simply click an X close button . For example, a modal that displays additional information about an instructor helping students choose a class, doesn't require buttons. Add .SimpleModal classname .
