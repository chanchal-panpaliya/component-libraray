Range Slider gives the end users an option to select a value within a configured range by dragging a slider.

Range sliders are used on web pages to allow the user to specify a numeric value that should not be less than a certain value and not greater than another given value.Sliders allow users to view and select a value (or range) in the range along a bar. Add .range-wrap as a base class, Add class .range to the input tag. To show slider value , Add .bubble to output tag.
